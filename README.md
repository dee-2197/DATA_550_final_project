# Generating Final report

To generate th final report, I use make command as there is a makefile
in my repo that describes targets, pre requists and action. it contains
targets to make demographics tables, boxplots and regression tables to
make final report. The make command uses code in code folder
04_render_report.R

# Geneating demogrphic tables

In the repo, code folder contains code- 01_make_table1.R to generate
demographic table for my report. Make output/table1.rds also generate
table in output folder named table1.rds

# Geneating figures

In the repo, code folder contains code- 02_make_graph.R to generate
boxplots for my report. Make output/boxplot.png also generate boxplot in
output folder named boxplot.png

# Geneating regression tables

In the repo, code folder contains code- 03_make_regression_tables.R to
generate regressipn table for my report. Make
output/regression_tables.rds also generate table in output folder named
regression_tables.rds
